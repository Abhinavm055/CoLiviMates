# 🏠 CO Space Finder

**CO Space Finder** is a modern web-based application that helps users **find roommates and shared housing spaces** efficiently.  
The platform connects **tenants, owners, and admins** through a clean and intuitive interface.

---

## 📌 Project Information

- **Project Name:** CO Space Finder  
- **Project Type:** Web Application  
- **Domain:** Housing & Roommate Finder  
- **Developed By:** M. Abhinav  
- **Role:** Student  

---

## 🎯 Project Objective

The goal of CO Space Finder is to simplify the process of finding **co-living spaces and compatible roommates** by providing:
- Easy-to-browse listings
- Secure user authentication
- Dedicated dashboards for different user roles

---

## 🚀 Features

- User Login & Signup
- Role-based Access (Tenant / Owner / Admin)
- Browse Housing & Roommate Listings
- Detailed Listing View
- Send & Manage Roommate Requests
- Tenant Dashboard
- Owner Dashboard for Listing Management
- Admin Panel for Monitoring and Control
- Responsive Design (Mobile & Desktop)
